## 音频级发音评估集成（免费开源）
- **Whisper word-level confidence**：利用 faster-whisper 内置的 `w.probability` 字段（每个词的 ASR 概率）
  - 低概率 = 发音不清晰，模型识别困难
  - 区分高频词（阈值0.55）、学术词（阈值0.45）、普通词（阈值0.60）
  - 智能提示生成：根据词中包含的 th/r/v/辅音丛等音素给出具体发音建议
- **Praat 音频声学分析**：`analyze_pronunciation_audio()` 新函数
  - 每词的 formant (F1/F2) 提取用于元音质量评估
  - 每词的 intensity CV（变异系数）用于清晰度评估
  - 全局 jitter/shimmer 用于声音质量评估
  - 输出 voice_quality 等级（clear / slightly unclear / needs improvement）
- **增强流畅度评估**：
  - 语速分6档（too_slow → too_fast）并给出具体 wpm 数值和建议
  - 停顿模式分析（自然停顿 vs 犹豫停顿）
  - 节奏一致性（articulation_rate / speech_rate 比值）
  - 语调丰富度分4级
  - CEFR 映射流利度等级（Elementary → Advanced）
- **LLM prompt 增强**：
  - 明确告知 LLM 发音问题来自音频级分析（ASR confidence），不仅是文本模式
  - 语速解读指引、发音矫正口腔提示
  - CRAA prompt 也传入发音问题数据
