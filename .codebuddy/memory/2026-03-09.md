## 实现 CRAA (Critical Response to Academic Arguments) 口语练习模式

基于 HKBU UE1 Module 4 PDF 文档的考试流程，为 AI Speech Coach 应用新增了完整的 CRAA 练习模式。

### 修改的文件

**后端：**
- `backend/app/database.py` — Exercise 模型新增 CRAA 字段：`argument_text`, `argument_audio_data`, `argument_audio_type`, `topic_context`, `key_claim`, `preparation_time`, `response_time`；`_run_migrations()` 添加自动迁移
- `backend/app/models/exercise.py` — 添加 `craa` ExerciseType 枚举，新增 CRAA 相关 Pydantic 字段
- `backend/app/routers/exercise_router.py` — 重构为使用 `_to_response` 辅助函数，添加 `POST/GET /{id}/argument-audio` 端点
- `backend/app/routers/practice_router.py` — 新增 `POST /craa-analyze` 端点，使用 CRAA Rubric 评估
- `backend/app/services/llm_service.py` — 新增 `generate_craa_feedback()` 函数
- `backend/app/prompts/craa_feedback.py` — 新建文件，CRAA 专用 LLM Prompt（三维度评分：Summary 40% + Counter 30% + Delivery 30%）

**前端：**
- `frontend/src/types/exercise.ts` — 添加 `craa` 类型和 CRAA 字段

- `frontend/src/pages/CRAAPracticePage.tsx` — 新建文件，完整4阶段CRAA练习流程：Intro → Listen → Prepare(倒计时) → Record(倒计时) → Analyzing → Result
